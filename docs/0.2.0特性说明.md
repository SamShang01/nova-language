# Nova语言 0.2.0 版本特性说明

## 版本概述

Nova语言 0.2.0 版本是一个重要的里程碑版本，引入了许多高级特性，包括泛型编程支持、特质（trait）系统、改进的模块系统、扩展的标准库功能和性能优化。

## 新增特性

### 1. 泛型编程支持

Nova语言现在支持泛型编程，允许编写更加通用和可复用的代码。

#### 1.1 泛型函数

```nova
generic func identity<T>(value: T) -> T {
    return value;
}

// 使用泛型函数
let int_value = identity(42);
let string_value = identity("hello");
```

#### 1.2 泛型结构体

```nova
generic struct Pair<T, U> {
    first: T;
    second: U;
}

// 使用泛型结构体
let pair = Pair<int, string>(first: 1, second: "hello");
```

#### 1.3 类型约束

```nova
generic func compare<T>(a: T, b: T) -> int
    where T: Comparable
{
    if a < b {
        return -1;
    } else if a > b {
        return 1;
    } else {
        return 0;
    }
}
```

### 2. 特质（Trait）系统

特质系统允许定义共享的行为，使代码更加模块化和可扩展。

#### 2.1 定义特质

```nova
trait Printable {
    func to_string() -> string;
}
```

#### 2.2 为类型实现特质

```nova
impl Printable for int {
    func to_string() -> string {
        return str(self);
    }
}

impl Printable for string {
    func to_string() -> string {
        return self;
    }
}
```

#### 2.3 使用特质约束

```nova
generic func print<T>(value: T)
    where T: Printable
{
    print(value.to_string());
}
```

### 3. 改进的模块系统

0.2.0版本引入了更完善的模块系统，支持更好的代码组织和依赖管理。

#### 3.1 模块声明

```nova
mod my_module;
```

#### 3.2 模块导入

```nova
use std::collections;
use std::math as math;
```

#### 3.3 模块搜索路径

模块系统支持自定义搜索路径，方便管理项目依赖。

### 4. 扩展的标准库功能

#### 4.1 数学模块

提供了丰富的数学函数和常量：

```nova
use std::math;

let angle = PI / 4;
let sin_value = sin(angle);
let cos_value = cos(angle);
let sqrt_value = sqrt(16);
```

支持的函数：
- 三角函数：`sin`, `cos`, `tan`
- 数学运算：`sqrt`, `pow`, `abs`, `floor`, `ceil`, `round`
- 比较函数：`min`, `max`
- 常量：`PI`, `E`

#### 4.2 字符串处理模块

提供了强大的字符串处理功能：

```nova
use std::string;

let s = "  Hello, World!  ";
let trimmed = trim(s);
let upper = to_upper(s);
let lower = to_lower(s);
let parts = split(s, ",");
```

支持的函数：
- 大小写转换：`to_upper`, `to_lower`
- 字符串操作：`trim`, `split`, `join`, `replace`, `substring`
- 字符串检查：`contains`, `starts_with`, `ends_with`

### 5. 性能优化

#### 5.1 常量折叠

编译器现在可以在编译时计算常量表达式，减少运行时计算：

```nova
// 编译时计算
let x = 1 + 2 + 3;  // 优化为 let x = 6;
let y = 2 * 3;      // 优化为 let y = 6;
```

#### 5.2 死代码消除

编译器会移除永远不会执行的代码，提高程序效率：

```nova
if false {
    // 这段代码会被移除
    print("never executed");
}
```

#### 5.3 优化器架构

引入了可扩展的优化器架构，支持添加自定义优化Pass：

```python
from nova.compiler.optimizer import Optimizer

optimizer = Optimizer()
optimized_ast = optimizer.optimize(ast)
```

## 修复的问题

### 1. Token类型问题
- 修复了TokenType枚举缺少TRUE和FALSE常量的问题
- 添加了泛型相关的Token类型（GENERIC, WHERE）

### 2. 标准库函数问题
- 修复了标准库函数无限递归的问题
- 改进了函数导入机制，避免与Python内置函数冲突

### 3. 版本管理问题
- 修复了版本导入路径问题
- 改进了版本判断逻辑

## 变更

### 1. 模块重命名
- 将async模块重命名为asynchronous，避免与Python保留关键字冲突

### 2. 版本号更新
- 版本号从0.1.0更新到0.2.0

## 兼容性

### 0.1.0到0.2.0迁移指南

大多数0.1.0的代码可以直接在0.2.0中运行，但需要注意以下几点：

1. 如果使用了async模块，需要更新为asynchronous模块
2. 泛型语法是新增功能，旧代码不受影响
3. 特质系统是新增功能，旧代码不受影响
4. 新增的标准库模块需要显式导入

## 示例程序

### 泛型示例

```nova
generic func swap<T>(a: T, b: T) -> (T, T) {
    return (b, a);
}

func main() {
    let x = 1;
    let y = 2;
    let (new_x, new_y) = swap(x, y);
    print(new_x);  // 输出 2
    print(new_y);  // 输出 1
}
```

### 特质示例

```nova
trait Display {
    func display() -> unit;
}

struct Point {
    x: int;
    y: int;
}

impl Display for Point {
    func display() -> unit {
        print(f"Point({self.x}, {self.y})");
    }
}

func main() {
    let p = Point(x: 10, y: 20);
    p.display();  // 输出 Point(10, 20)
}
```

### 数学模块示例

```nova
use std::math;

func main() {
    let angle = PI / 6;
    print(sin(angle));   // 输出 0.5
    print(cos(angle));   // 输出 0.8660254037844386
    print(sqrt(25));    // 输出 5.0
    print(pow(2, 10));  // 输出 1024.0
}
```

### 字符串处理示例

```nova
use std::string;

func main() {
    let s = "Hello, World!";
    print(to_upper(s));      // 输出 HELLO, WORLD!
    print(to_lower(s));      // 输出 hello, world!
    print(trim("  test  ")); // 输出 test
    print(split("a,b,c", ",")); // 输出 ["a", "b", "c"]
}
```

## 性能提升

0.2.0版本通过以下优化提升了性能：

1. **常量折叠**：减少了运行时计算开销
2. **死代码消除**：减少了代码大小和执行时间
3. **改进的模块系统**：优化了模块加载和符号解析
4. **标准库优化**：改进了常用函数的实现

## 未来计划

虽然0.2.0版本引入了许多重要特性，但仍有改进空间：

1. **完整的泛型支持**：当前版本支持基本泛型，未来将支持更复杂的泛型约束和变体
2. **完整的特质系统**：当前版本支持基本的特质定义和实现，未来将支持特质继承和默认实现
3. **JIT编译器**：计划引入JIT编译器，进一步提升性能
4. **更多标准库功能**：计划添加更多实用的标准库模块
5. **完善的错误处理**：改进错误信息和调试工具

Nova语言 0.2.0 版本是一个功能丰富的版本，引入了泛型编程、特质系统、改进的模块系统、扩展的标准库功能和性能优化。这些特性使Nova语言更加强大和易用，为未来的发展奠定了坚实的基础。
