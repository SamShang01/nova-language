# Nova 语言更新日志

## [0.9.4] - 2026-03-01
### 修复
- **运算符重载语法糖**：
  - 修复 `v1 + v2` 语法糖，现在会自动调用 `v1.__add__(v2)`
  - 修复 `v1 - v2` 语法糖，现在会自动调用 `v1.__sub__(v2)`
  - 修复 `v1 == v2` 语法糖，现在会自动调用 `v1.__eq__(v2)`
  - 修复方法调用时参数绑定错误，`this` 参数不再被重复绑定到第一个形参

### 新增
- **NovaInstance 运算符方法**：
  - 添加 `__sub__` 方法支持减法运算
  - 添加 `__eq__` 方法支持相等比较

## [0.9.3] - 2026-03-01
### 新增
- **完整的面向对象编程支持**：
  - **构造函数**：支持 `init()` 方法初始化类实例
  - **静态成员**：支持 `static` 关键字定义静态字段和方法
  - **抽象类和抽象方法**：支持 `abstract` 关键字定义抽象类和抽象方法
  - **多态和方法重写**：支持子类重写父类方法，实现动态绑定
  - **访问修饰符**：支持 `private`、`protected`、`public` 访问修饰符
  - **super 关键字**：支持通过 `super` 调用父类方法
  - **属性访问器**：支持 getter 和 setter 方法
  - **运算符重载**：支持 `__add__`、`__eq__` 等特殊方法

## [0.9.2] - 2026-02-27
### 修复
- **const 常量赋值修复**：
  - 修复 `const` 声明的常量可以被重新赋值的问题
  - 现在尝试给 `const` 常量赋值会报错："Cannot assign to constant 'x'"
  - 同时 `let` 声明的不可变变量也不能被重新赋值

- **延迟运算特性控制**：
  - 修复 `DeferredOperations` 特性控制逻辑
  - 未启用特性时：`0.1+0.2-0.2` 返回 `0.10000000000000003`（保留浮点数精度问题）
  - 启用特性时：`from __future__ import DeferredOperations;` + `0.1+0.2-0.2` 返回 `0.1`（优化后）

## [0.9.1] - 2026-02-27
### 新增
- **延迟运算功能**：
  - 实现对乘法和除法操作的延迟运算，避免浮点数精度问题
  - 例如：1/3*3 现在会正确计算为 1.0，而不是 0.9999999999999999
  - 优化计算顺序，先处理所有乘法，再处理除法

### 修复
- **运算符重载修复**：
  - 修复结构体运算符重载时的无限递归问题
  - 修复BINARY_ADD指令的实现，正确处理基本类型和自定义类型
  - 修复代码生成器对成员表达式的处理，正确生成访问成员的代码
  - 修复NovaFunction的__call__方法，移除硬编码的__add__处理逻辑
  - 支持在结构体方法中访问self参数和结构体名称
- **词法分析器修复**：
  - 修复*字符解析为STAR的问题，正确解析为MULTIPLY
