# Nova 0.9.0 版本说明

## 版本信息
- 版本号：0.9.0
- 发布日期：2026-02-27

## 主要特性

### 运算符重载支持
本版本实现了结构体的运算符重载功能，允许用户为自定义类型定义运算符的行为。

#### 支持的运算符
- `__add__`：加法运算符（+）
- `__sub__`：减法运算符（-）
- `__mul__`：乘法运算符（*）
- `__div__`：除法运算符（/）
- `__eq__`：等于运算符（==）
- `__ne__`：不等于运算符（!=）
- `__lt__`：小于运算符（<）
- `__le__`：小于等于运算符（<=）
- `__gt__`：大于运算符（>）
- `__ge__`：大于等于运算符（>=）

#### 使用示例

```nova
struct Vector {
    x: float,
    y: float,
    
    func __add__(self: Vector, other: Vector) -> Vector {
        return Vector(x: self.x + other.x, y: self.y + other.y);
    }
}

func main() {
    let v1 = Vector(x: 1.0, y: 2.0);
    let v2 = Vector(x: 3.0, y: 4.0);
    let v3 = v1 + v2;
    print("v1 + v2 = (" + str(v3.x) + ", " + str(v3.y) + ")");
}
```

## 技术实现

### 语义分析
- 在语义分析阶段，识别结构体中的特殊方法（如`__add__`）
- 为结构体方法添加self参数和结构体名称到方法作用域
- 支持在方法中访问self参数和结构体名称

### 代码生成
- 为结构体定义生成NovaClass对象
- 为结构体方法生成NovaFunction对象
- 支持成员表达式的代码生成

### 虚拟机执行
- 实现NovaClass和NovaInstance类，支持结构体的实例化
- 实现NovaFunction类，支持函数调用
- 实现BINARY_ADD等二元运算指令，支持运算符重载

## 已知问题
本版本存在以下已知问题，将在后续版本中修复：
- 结构体运算符重载时可能出现无限递归问题
- 成员表达式的代码生成不完整
- NovaFunction的__call__方法存在硬编码逻辑

## 兼容性
- 本版本向后兼容0.8.0及更高版本
- 建议使用Python 3.7或更高版本
