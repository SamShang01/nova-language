# Nova语言 v1.0.0

这是Nova语言的第一个正式版本。

## 版本信息

- 版本号: 1.0.0
- 发布日期: 2026-03-12
- 状态: 正式发布

## 核心特性

1. 完整的编译器系统（词法、语法、语义分析）
2. 强大的类型系统（泛型、Trait、自定义类型）
3. 丰富的标准库（IO、数学、字符串、集合）
4. 开发工具（REPL、VS Code插件、Nova IDE）
5. 编译优化（增量编译、并行编译、LLVM JIT）
6. 包管理系统

## 安装

```bash
pip install -e .
```

## 使用

```bash
# 运行Nova文件
nova run hello.nova

# 进入REPL
nova repl

# 查看帮助
nova --help
```
