# Changelog

## [1.2.0] - 2026-04-02

### 🎉 Nova语言 1.2.0 版本发布！

这是Nova语言的第二个次要版本更新，带来了更多功能改进和性能优化。

### ✨ 新特性

1. **增强的类型系统**
   - 改进类型推断算法
   - 增强泛型类型支持
   - 完善类型检查机制

2. **扩展的标准库**
   - `argparse`模块：命令行参数解析
   - `json`模块：JSON数据处理
   - `re`模块：正则表达式支持

3. **编译器优化**
   - 改进编译速度
   - 优化生成的字节码
   - 增强错误提示

4. **开发工具增强**
   - 改进REPL交互体验
   - 增强调试器功能
   - 优化包管理系统

5. **文档完善**
   - 新增多语言文档
   - 完善API文档
   - 添加更多示例代码

### 🐛 修复内容

- 修复了编译器中的一些边缘情况
- 解决了运行时的内存管理问题
- 修复了标准库中的一些bug

### 📚 文档更新

- 更新了语言参考文档
- 添加了更多示例代码
- 改进了API文档

### 🔒 兼容性

- 完全向后兼容1.1.0版本
- 所有现有代码无需修改即可运行
