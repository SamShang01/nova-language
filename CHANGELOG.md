# Changelog

所有版本的更新日志都记录在 [CHANGELOG_ALL.md](CHANGELOG_ALL.md) 中。

## [1.1.0] - 2026-03-25

### 🎉 Nova语言 1.1.0 版本发布！

这是Nova语言的第一个次要版本更新，带来了多项功能改进和性能优化。

### ✨ 新特性

1. **增强的STL库**
   - 新增`trie`数据结构，用于高效字符串查找
   - 改进`vector`和`list`的性能
   - 添加更多算法函数

2. **扩展的标准库**
   - `crypto`模块：添加基本的加密功能
   - `network`模块：新增HTTP和Socket支持
   - `time`模块：增强时间处理功能

3. **语言特性改进**
   - 支持更多的操作符重载
   - 改进类型推断系统
   - 增强错误处理机制

4. **性能优化**
   - 虚拟机执行速度提升
   - 编译时间减少
   - 内存使用优化

5. **开发工具改进**
   - REPL交互体验优化
   - 调试器功能增强
   - 包管理系统改进

### 🐛 修复内容

- 修复了类型检查中的一些边缘情况
- 解决了编译过程中的内存泄漏问题
- 修复了标准库中的一些bug

### 📚 文档更新

- 更新了语言参考文档
- 添加了更多示例代码
- 改进了API文档

### 🔒 兼容性

- 完全向后兼容1.0.0版本
- 所有现有代码无需修改即可运行
