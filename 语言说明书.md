# Nova语言说明书

## 1. 语言简介

Nova是一种现代化的编程语言，设计目标是提供简洁、优雅且高效的编程体验。它融合了多种现代编程语言的优点，同时保持了自身的独特性。

### 1.1 设计理念

- **简洁优雅**：语法简洁明了，易于阅读和编写
- **类型安全**：支持静态类型检查，减少运行时错误
- **高性能**：采用虚拟机执行，优化执行效率
- **模块化**：支持模块化编程，便于代码组织和复用
- **异步支持**：内置异步编程支持，简化并发编程

### 1.2 应用场景

- 后端服务开发
- 脚本编写
- 数据处理
- 教育学习

## 2. 语法基础

### 2.1 变量声明

Nova语言支持三种变量声明方式：`let`、`var` 和 `const`。

```nova
// 不可变变量（推荐）
let x = 10;

// 可变变量
var y = 20;
y = 30; // 合法

// 常量
const PI = 3.14159;
PI = 3.14; // 编译错误
```

### 2.2 数据类型

Nova语言支持以下基本数据类型：

- **整数** (`int`)：如 `1`, `42`, `-10`
- **浮点数** (`float`)：如 `3.14`, `2.718`, `-0.5`
- **字符串** (`string`)：如 `"hello"`, `"world"`
- **布尔值** (`bool`)：如 `true`, `false`
- **字符** (`char`)：如 `'a'`, `'b'`
- **单元值** (`unit`)：表示无返回值，如 `()`

### 2.3 表达式

#### 2.3.1 算术表达式

```nova
let a = 1 + 2;     // 加法
let b = 3 - 4;     // 减法
let c = 5 * 6;     // 乘法
let d = 7 / 8;     // 除法
let e = 9 % 10;    // 取模
```

#### 2.3.2 比较表达式

```nova
let a = 1 == 2;    // 等于
let b = 3 != 4;    // 不等于
let c = 5 < 6;     // 小于
let d = 7 <= 8;    // 小于等于
let e = 9 > 10;    // 大于
let f = 11 >= 12;  // 大于等于
```

#### 2.3.3 逻辑表达式

```nova
let a = true && false; // 逻辑与
let b = true || false; // 逻辑或
let c = !true;         // 逻辑非
```

### 2.4 控制流

#### 2.4.1 条件语句

```nova
if x > 0 {
    print("x is positive");
} else if x < 0 {
    print("x is negative");
} else {
    print("x is zero");
}
```

#### 2.4.2 循环语句

```nova
// for循环
for i in 0..10 {
    print(i);
}

// while循环
var i = 0;
while i < 10 {
    print(i);
    i = i + 1;
}

// loop循环（无限循环）
loop {
    print("Hello");
    break; // 跳出循环
}
```

### 2.5 函数定义

```nova
func add(a: int, b: int) -> int {
    return a + b;
}

// 调用函数
let result = add(1, 2);
print(result); // 输出 3
```

### 2.6 结构体

```nova
struct Point {
    x: int;
    y: int;
}

// 创建结构体实例
let p = Point(x: 10, y: 20);

// 访问字段
print(p.x); // 输出 10
print(p.y); // 输出 20
```

### 2.7 枚举

```nova
enum Color {
    Red;
    Green;
    Blue;
}

// 使用枚举
let c = Color::Red;
```

### 2.8 异步编程

```nova
async func fetch_data() -> string {
    // 模拟异步操作
    await sleep(1000);
    return "data";
}

// 调用异步函数
async func main() {
    let data = await fetch_data();
    print(data);
}
```

## 3. 标准库

### 3.1 核心模块 (`core`)

核心模块提供了Nova语言的基本功能，包括：

- **基本类型**：`int`, `float`, `string`, `bool`, `char`, `unit`
- **内置函数**：
  - `print(value)`：打印值
  - `len(value)`：获取长度
  - `input(prompt)`：获取用户输入
  - `range(start, end, step)`：生成范围
  - `enumerate(iterable)`：枚举可迭代对象

### 3.2 集合模块 (`collections`)

集合模块提供了常用的数据结构，包括：

- **列表** (`List`)：有序可变序列
- **字典** (`Dict`)：键值对映射
- **集合** (`Set`)：无序唯一元素集合

### 3.3 IO模块 (`io`)

IO模块提供了输入输出功能，包括：

- **文件操作**：`File` 类，支持文件读写
- **标准流**：`stdin`, `stdout`, `stderr`

### 3.4 异步模块 (`asynchronous`)

异步模块提供了异步编程支持，包括：

- **Future**：表示异步操作的结果
- **async/await**：异步函数和等待操作

## 4. 版本管理

Nova语言采用语义化版本管理，版本号格式为 `x.y.z`，其中：

- `x`：主版本号，重大变更
- `y`：次版本号，新功能
- `z`：修订号， bug 修复

### 4.1 版本判断

在代码中，可以使用 `__version__` 变量来判断当前Nova语言的版本：

```nova
if __version__ >= (0, 2, 0) {
    // 使用 0.2.0 及以上版本的功能
} else {
    // 使用兼容旧版本的代码
}
```

### 4.2 版本安装

可以使用 `install` 命令安装指定版本的Nova语言：

```bash
python script.py install --version==0.1.0
```

## 5. 编译器架构

Nova语言的编译器采用传统的编译器架构，分为以下几个阶段：

### 5.1 词法分析

词法分析器将源代码转换为Token序列。

### 5.2 语法分析

语法分析器将Token序列转换为抽象语法树（AST）。

### 5.3 语义分析

语义分析器对AST进行类型检查和作用域分析。

### 5.4 代码生成

代码生成器将AST转换为虚拟机指令。

### 5.5 虚拟机执行

虚拟机执行生成的指令。

## 8. 常见问题

### 8.1 变量不可变

Nova语言中，使用 `let` 声明的变量是不可变的，这是一个设计选择，旨在减少错误。如果需要可变变量，请使用 `var`。

### 8.2 类型错误

Nova语言是类型安全的，编译器会检查类型错误。如果遇到类型错误，请检查变量类型和函数参数类型是否匹配。

### 8.3 异步函数

异步函数必须使用 `async` 关键字声明，调用异步函数时必须使用 `await` 关键字。
