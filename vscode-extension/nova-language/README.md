# Nova Language VS Code Extension

Nova language support for Visual Studio Code.

## Features

- Syntax highlighting for Nova language
- Code snippets for common constructs
- Basic language configuration

## Installation

1. Download the VSIX file from the releases page
2. Open VS Code
3. Click on the Extensions icon
4. Click on "Install from VSIX..."
5. Select the downloaded VSIX file

## Usage

- Create a new file with `.nova` extension
- Start writing Nova code
- Use code snippets by typing the prefix (e.g., `func` for function)

## Supported Snippets

- `func` - Create a function
- `struct` - Create a struct
- `if` - Create an if statement
- `while` - Create a while loop
- `for` - Create a for loop
- `let` - Create a variable declaration
