Bienvenue dans Nova

Nova est un langage de programmation moderne et statiquement typé conçu pour créer des logiciels fiables et efficaces. Il combine les meilleures fonctionnalités de langages comme Python, TypeScript et Rust, offrant une syntaxe claire, un système de types puissant et des fonctionnalités avancées.

Typage statique

La vérification des types au moment de la compilation permet de détecter les erreurs tôt et d'améliorer les performances.

Syntaxe claire

Une syntaxe de type Python, facile à lire et à écrire.

Orienté objet

Supporte les classes, l'héritage et le polymorphisme.

Fonctionnalités fonctionnelles

Supporte les fonctions lambda, les fonctions d'ordre supérieur et l'immuabilité.

Traits

Réutilisation du code par les traits pour une composition flexible.

Génériques

Écrivez du code générique sécurisé en termes de types pour une réutilisation maximale.

Pour commencer

Prêt à commencer à utiliser Nova ? Suivez ces étapes :

  1. Installez Nova sur votre système
  2. Lisez le tutoriel Hello World pour écrire votre premier programme
  3. Explorez le tutoriel complet pour en apprendre plus sur les fonctionnalités
  4. Consultez les exemples pour des cas d'utilisation pratiques
  5. Référez-vous à la documentation de référence pour des informations détaillées

Besoin d'aide ?

Si vous avez des questions ou besoin d'assistance, consultez la FAQ ou rejoignez les forums de la communauté Nova.

Communauté

Nova dispose d'une communauté croissante de développeurs passionnés par le langage. Voici comment vous pouvez participer :

  • Rejoignez le serveur Discord pour discuter avec d'autres développeurs Nova
  • Suivez-nous sur GitHub pour contribuer au projet
  • Signalez des bugs et suggérez des fonctionnalités via les problèmes GitHub
  • Écrivez des articles de blog ou des tutoriels sur Nova
