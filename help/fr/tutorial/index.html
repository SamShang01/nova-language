Bienvenue dans le tutoriel Nova

Ce tutoriel vous guidera à travers les bases du langage de programmation Nova, de l'installation aux sujets avancés. Suivez les étapes dans l'ordre pour construire une base solide.

Pour commencer

Nous recommandons de suivre le tutoriel dans l'ordre ci-dessous pour tirer le meilleur parti de votre expérience d'apprentissage.

  • Classes

  • Collections

  • Gestion des erreurs

  • Entrée/Sortie de fichiers

    Apprenez comment lire et écrire des fichiers dans Nova.
