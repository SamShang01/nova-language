Introduction to Control Flow

Control flow statements allow you to control the order in which code is executed. Nova provides several control flow statements, including conditional statements and loops.

Conditional Statements

if Statement

The if statement is used to execute code only if a condition is true:

let age = 18; if (age >= 18) { print("You are an adult"); }

if-else Statement

The if-else statement allows you to execute different code based on whether a condition is true or false:

let age = 16; if (age >= 18) { print("You are an adult"); } else { print("You are a minor"); }

if-else if-else Statement

You can chain multiple conditions using else if:

let score = 85; if (score >= 90) { print("A"); } else if (score >= 80) { print("B"); } else if (score >= 70) { print("C"); } else if (score >= 60) { print("D"); } else { print("F"); }

Switch Statement

The switch statement allows you to select one of several code blocks to execute based on the value of an expression:

let day = "Monday"; switch (day) { case "Monday": print("Start of the week"); break; case "Friday": print("End of the work week"); break; case "Saturday": case "Sunday": print("Weekend"); break; default: print("Midweek"); break; }

Note

In Nova, the break statement is required after each case to prevent fall-through. If you want fall-through behavior, you can omit the break statement.

Loops

for Loop

The for loop is used to execute a block of code a specified number of times:

// Traditional for loop for (let i = 0; i < 5; i++) { print(i); } // Output: 0, 1, 2, 3, 4

for-in Loop

The for-in loop is used to iterate over elements in a collection:

// Iterate over an array let fruits = ["apple", "banana", "cherry"]; for (let fruit in fruits) { print(fruit); } // Output: apple, banana, cherry // Iterate over a string let message = "Hello"; for (let char in message) { print(char); } // Output: H, e, l, l, o

while Loop

The while loop executes a block of code as long as a condition is true:

let count = 0; while (count < 5) { print(count); count++; } // Output: 0, 1, 2, 3, 4

do-while Loop

The do-while loop executes a block of code at least once, and then repeatedly executes it as long as a condition is true:

let count = 0; do { print(count); count++; } while (count < 5); // Output: 0, 1, 2, 3, 4

Loop Control Statements

break Statement

The break statement is used to exit a loop early:

for (let i = 0; i < 10; i++) { if (i === 5) { break; // Exit the loop when i is 5 } print(i); } // Output: 0, 1, 2, 3, 4

continue Statement

The continue statement is used to skip the current iteration of a loop and continue with the next iteration:

for (let i = 0; i < 10; i++) { if (i % 2 === 0) { continue; // Skip even numbers } print(i); } // Output: 1, 3, 5, 7, 9

return Statement

The return statement is used to exit a function and return a value:

def find_first_negative(numbers) { for (let number in numbers) { if (number < 0) { return number; // Return the first negative number } } return null; // Return null if no negative numbers found } let numbers = [1, 2, -3, 4, -5]; let result = find_first_negative(numbers); print(result); // Output: -3

Nested Control Flow

You can nest control flow statements inside each other:

// Nested loops for (let i = 0; i < 3; i++) { print("Outer loop: " + i); for (let j = 0; j < 2; j++) { print(" Inner loop: " + j); } } // Nested if statements let age = 25; let has_license = true; if (age >= 18) { if (has_license) { print("You can drive"); } else { print("You need a license to drive"); } } else { print("You are too young to drive"); }

Logical Operators

Logical operators are often used in conditional statements to combine multiple conditions:

AND Operator (&&)

let age = 25; let has_license = true; if (age >= 18 && has_license) { print("You can drive"); }

OR Operator (||)

let is_weekend = true; let is_vacation = false; if (is_weekend || is_vacation) { print("No work today!"); }

NOT Operator (!)

let is_raining = false; if (!is_raining) { print("It's a nice day!"); }

Comparison Operators

Comparison operators are used to compare values in conditional statements:

  • == - Equal to
  • != - Not equal to
  • < - Less than
  • <= - Less than or equal to
  • > - Greater than
  • >= - Greater than or equal to
  • === - Strict equal to (checks type and value)
  • !== - Strict not equal to (checks type and value)
let a = 5; let b = "5"; print(a == b); // true (loose equality) print(a === b); // false (strict equality - different types)

Best Practices

  • Keep it simple: Avoid overly complex nested control flow.
  • Use meaningful variable names: Make your conditions easy to understand.
  • Indent properly: Use consistent indentation to make your code more readable.
  • Use switch for multiple conditions: Use switch statements instead of long if-else if chains when appropriate.
  • Avoid infinite loops: Make sure your loops have a way to terminate.

Common Mistakes

  • Using assignment instead of equality: Remember to use == or === for comparisons, not =.
  • Infinite loops: Forgetting to update the loop variable in a while loop can lead to infinite loops.
  • Misusing break and continue: Make sure you're using these statements correctly.
  • Complex nested conditions: Too many nested if statements can make code hard to read.
