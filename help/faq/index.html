General Questions

What is Nova programming language?

Nova is a modern, statically-typed programming language designed for building reliable and efficient software. It combines the best features of languages like Python, TypeScript, and Rust, offering a clean syntax, strong type system, and powerful features.

Who created Nova?

Nova was created by a team of experienced software engineers who wanted to build a language that addresses the limitations of existing languages while providing a modern, developer-friendly experience.

Is Nova open source?

Yes, Nova is an open-source project. You can find the source code on GitHub and contribute to its development.

What platforms does Nova support?

Nova is designed to be cross-platform and supports Windows, macOS, and Linux. You can write code once and run it on any supported platform.

Installation

How do I install Nova?

You can install Nova using the package manager for your operating system, or by downloading the installer from the official website. Detailed installation instructions are available in the Installation tutorial.

What are the system requirements for Nova?

Nova has minimal system requirements. You need a computer with at least 2GB of RAM and 500MB of free disk space. Nova runs on Windows 10+, macOS 10.15+, and most Linux distributions.

How do I check if Nova is installed correctly?

Open a terminal and run nova --version. If Nova is installed correctly, you should see the version number displayed.

Language Features

Is Nova a statically-typed language?

Yes, Nova is a statically-typed language. This means that type checking is done at compile time, which helps catch errors early and improves performance.

Does Nova support object-oriented programming?

Yes, Nova supports object-oriented programming with classes, inheritance, and polymorphism. It also supports other programming paradigms like functional programming.

What is the difference between Nova and Python?

While Nova has a syntax similar to Python, it offers static typing, better performance, and additional features like traits and generics. Nova also has a more modern type system and better error handling.

Does Nova support concurrency?

Yes, Nova supports concurrency through threads and async/await syntax. It also provides safe concurrency primitives to help avoid common issues like race conditions.

Development

What editors and IDEs support Nova?

Nova has support for popular editors and IDEs like Visual Studio Code, IntelliJ IDEA, and Sublime Text. You can install extensions or plugins to get syntax highlighting, code completion, and other features.

How do I run a Nova program?

You can run a Nova program using the nova command followed by the filename. For example: nova hello.nova.

How do I debug a Nova program?

Nova supports debugging through IDEs that have Nova plugins, or you can use the built-in --debug flag to get more detailed error messages.

How do I create a Nova module?

To create a Nova module, simply create a .nova file with your code and use the export keyword to expose functions, classes, or variables. You can then import them in other files using the import keyword.

Standard Library

What's included in Nova's standard library?

Nova's standard library includes modules for common tasks like file I/O, networking, collections, date/time handling, and more. You can find detailed documentation in the Library documentation.

How do I use the standard library?

You can import standard library modules using the import keyword. For example: import std.io to use the I/O module.

Can I create my own libraries?

Yes, you can create your own libraries and distribute them as packages. Nova has a package manager that allows you to publish and install packages.

Troubleshooting

I'm getting a compile error. What should I do?

Check the error message carefully. It should tell you what's wrong and where the error is. Common issues include syntax errors, type mismatches, and undefined variables. If you're still stuck, consult the reference documentation or ask for help in the Nova community.

My program is running slowly. How can I optimize it?

There are several ways to optimize Nova programs: use efficient algorithms, avoid unnecessary computations, use appropriate data structures, and take advantage of Nova's performance features like static typing and inline functions. You can also use profiling tools to identify bottlenecks.

How do I handle errors in Nova?

Nova uses try-catch statements for error handling. You can catch specific exceptions or use a general catch block. You can also create custom exception classes for your application.

Future Plans
