Virtual Machine Instructions

Nova's virtual machine uses a set of instructions to execute code. Here's a reference for these instructions:

// Arithmetic instructions BINARY_ADD // Add two values BINARY_SUB // Subtract two values BINARY_MUL // Multiply two values BINARY_DIV // Divide two values BINARY_MOD // Modulo operation // Comparison instructions COMPARE_EQ // Equal to COMPARE_NE // Not equal to COMPARE_LT // Less than COMPARE_LE // Less than or equal to COMPARE_GT // Greater than COMPARE_GE // Greater than or equal to // Control flow instructions JUMP // Unconditional jump JUMP_IF_TRUE // Jump if value is true JUMP_IF_FALSE // Jump if value is false // Stack instructions POP // Pop value from stack DUP // Duplicate top stack value SWAP // Swap top two stack values // Variable instructions LOAD_NAME // Load variable by name STORE_NAME // Store variable by name LOAD_GLOBAL // Load global variable STORE_GLOBAL // Store global variable // Function instructions CALL // Call function RETURN_VALUE // Return from function // Object instructions LOAD_ATTR // Load attribute from object STORE_ATTR // Store attribute in object // Collection instructions BUILD_LIST // Build list from stack values BUILD_TUPLE // Build tuple from stack values BUILD_DICT // Build dictionary from stack values // Other instructions NOP // No operation HALT // Halt execution

Built-in Functions

Nova provides a set of built-in functions that are available globally:

// Print functions print(value) // Print value to console println(value) // Print value with newline // Type functions typeof(value) // Return type of value isinstance(value, type) // Check if value is instance of type // Conversion functions str(value) // Convert to string int(value) // Convert to integer float(value) // Convert to float bool(value) // Convert to boolean // Collection functions len(collection) // Return length of collection // Math functions abs(value) // Absolute value round(value) // Round to nearest integer // String functions range(start, end, step) // Generate range of numbers // Error functions throw(error) // Throw exception
