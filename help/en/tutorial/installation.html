Installation Guide

Install Nova on your system

Home > Tutorial > Installation

System Requirements

Nova is designed to run on a variety of systems. Here are the minimum requirements:

  • Operating System: Windows 10+, macOS 10.15+, Linux (Ubuntu 18.04+, Fedora 30+)
  • CPU: 1 GHz or faster processor
  • Memory: 2 GB RAM
  • Storage: 100 MB of free space
  • Python: 3.7 or higher (required for runtime)

Installation Methods

Windows Installation

  1. Download the installer: Visit the Nova website and download the Windows installer (.exe) for your system architecture (32-bit or 64-bit).
  2. Run the installer: Double-click the downloaded .exe file and follow the on-screen instructions.
  3. Add to PATH: During installation, select the option to add Nova to your system PATH.
  4. Verify installation: Open a new command prompt and run nova --version to confirm the installation was successful.

macOS Installation

  1. Download the package: Visit the Nova website and download the macOS package (.pkg).
  2. Run the installer: Double-click the downloaded .pkg file and follow the on-screen instructions.
  3. Add to PATH: The installer will automatically add Nova to your PATH.
  4. Verify installation: Open a new terminal window and run nova --version to confirm the installation was successful.

Linux Installation

Using Package Manager (Debian/Ubuntu)
# Add Nova repository sudo apt update sudo apt install wget gnupg wget -qO- https://nova-lang.org/keys/nova.gpg | sudo apt-key add - echo "deb https://nova-lang.org/repos/deb stable main" | sudo tee /etc/apt/sources.list.d/nova.list # Install Nova sudo apt update sudo apt install nova
Using Package Manager (Fedora)
# Add Nova repository sudo dnf config-manager --add-repo https://nova-lang.org/repos/rpm/nova.repo # Install Nova sudo dnf install nova
From Source
  1. Clone the repository: git clone https://github.com/nova-lang/nova.git
  2. Navigate to the directory: cd nova
  3. Install dependencies: pip install -r requirements.txt
  4. Install Nova: python setup.py install
  5. Verify installation: nova --version

Alternative Installation Methods

Using pip

You can also install Nova using pip:

pip install nova-lang

From Source

For advanced users who want to build from source:

  1. Clone the repository: git clone https://github.com/nova-lang/nova.git
  2. Navigate to the directory: cd nova
  3. Install dependencies: pip install -r requirements.txt
  4. Build Nova: python setup.py build
  5. Install Nova: python setup.py install

Verifying Installation

To verify that Nova has been installed correctly, open a terminal or command prompt and run:

nova --version

You should see the version number of Nova displayed. You can also run the REPL (Read-Eval-Print Loop) by typing:

nova

This will start the Nova interactive shell, where you can test simple expressions.

Troubleshooting

Common Issues

  • Nova command not found: Make sure Nova is added to your system PATH. On Windows, you may need to restart your command prompt. On Linux/macOS, you may need to run source ~/.bashrc or source ~/.zshrc.
  • Python dependency issues: Ensure you have Python 3.7 or higher installed. You can check your Python version with python --version.
  • Permission errors: On Linux/macOS, you may need to use sudo when installing Nova.
