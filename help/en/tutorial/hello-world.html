Introduction

Welcome to your first Nova program! In this tutorial, you'll learn how to create and run a simple "Hello, World!" program in Nova. This is the traditional first step for learning any programming language.

Step 1: Create a Nova File

First, create a new file with a .nova extension. You can name it whatever you like, but for this tutorial, let's call it hello.nova.

Windows

  1. Open Notepad or your favorite text editor
  2. Save the file as hello.nova (make sure to select "All Files" in the save dialog)

macOS/Linux

  1. Open your favorite text editor (e.g., VS Code, Sublime Text, or nano)
  2. Save the file as hello.nova

Step 2: Write the Hello World Program

Now, let's write the code for our Hello World program. In Nova, the simplest way to print something to the console is to use the print function.

def main() { print("Hello, World!"); }

Let's break down what this code does:

  • def main() { - This defines a function named main. In Nova, the main function is the entry point of your program.
  • print("Hello, World!"); - This calls the print function with the string "Hello, World!" as an argument. This will display the text on the console.
  • } - This closes the main function.

Step 3: Run the Program

Now that we've written our program, let's run it. Open a terminal or command prompt and navigate to the directory where you saved hello.nova.

Running the Program

# Run the program using the nova command nova hello.nova

You should see the output:

Hello, World!

Alternative: Using the REPL

You can also run Nova code interactively using the REPL (Read-Eval-Print Loop). To start the REPL, simply type nova in your terminal:

$ nova Nova 1.0.0 (REPL mode) Type 'exit' to quit. > print("Hello, World!"); Hello, World! >

To exit the REPL, type exit or press Ctrl+D (Ctrl+Z on Windows).

Understanding the Code

Let's take a closer look at the Hello World program:

Function Definition

In Nova, functions are defined using the def keyword, followed by the function name, parentheses for parameters, and a block of code enclosed in curly braces.

The main Function

The main function is special in Nova. When you run a Nova program, the interpreter looks for a main function and calls it automatically. This is the entry point of your program.

The print Function

The print function is a built-in function in Nova that displays text to the console. It takes one or more arguments and prints them separated by spaces.

Variations

Here are a few variations of the Hello World program to try:

Printing Multiple Lines

def main() { print("Hello, World!"); print("Welcome to Nova!"); }

Using Variables

def main() { let message = "Hello, World!"; print(message); }

Printing with Variables

def main() { let name = "World"; print("Hello, " + name + "!"); }

Common Errors

Here are some common errors you might encounter when writing your first Nova program:

  • Syntax Error: Forgetting to close parentheses or curly braces. Make sure all opening parentheses, brackets, and braces have corresponding closing ones.
  • Name Error: Using a variable or function that hasn't been defined. Check your spelling and make sure all names are correct.
  • File Not Found: Running the program from the wrong directory. Make sure you're in the same directory as your .nova file.
