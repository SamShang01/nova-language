What is Nova?

Nova is a modern programming language designed to be expressive, efficient, and easy to learn. It combines the best features of traditional languages with modern innovations, making it suitable for a wide range of applications from web development to data science.

Key Features

Modern Syntax

Clean, readable syntax that reduces boilerplate code and makes programming more enjoyable.

Static Typing

Optional type annotations for better code reliability and tooling support.

Object-Oriented

Full support for classes, inheritance, and polymorphism.

Functional Programming

First-class functions and lambda expressions for functional programming paradigms.

Generics

Type-safe generic programming for reusable code.

Error Handling

Comprehensive error handling with try-catch blocks.

Hello World Example

Let's start with a simple Hello World program in Nova:

def main() { print("Hello, World!"); }

That's it! A simple and clean syntax that's easy to read and understand.

Basic Syntax Examples

Variables

// Variable declaration let x = 10; let name = "Nova"; let is_active = true;

Functions

// Function definition def add(a, b) { return a + b; } // Function call let result = add(5, 3); print(result); // Output: 8

Control Flow

// If-else statement if (x > 0) { print("Positive"); } else if (x < 0) { print("Negative"); } else { print("Zero"); } // Loop for (let i = 0; i < 5; i++) { print(i); }

Why Choose Nova?

Nova was designed with the following principles in mind:

  • Readability: Code should be easy to read and understand
  • Expressiveness: Write less code to accomplish more
  • Reliability: Catch errors early with static typing
  • Performance: Efficient execution for both small and large programs
  • Flexibility: Adapt to different programming styles and paradigms
