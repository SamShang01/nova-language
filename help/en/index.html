Welcome to Nova

Nova is a modern, statically-typed programming language designed for building reliable and efficient software. It combines the best features of languages like Python, TypeScript, and Rust, offering a clean syntax, strong type system, and powerful features.

Static Typing

Compile-time type checking helps catch errors early and improves performance.

Clean Syntax

Python-like syntax that's easy to read and write.

Object-Oriented

Supports classes, inheritance, and polymorphism.

Functional Features

Supports lambda functions, higher-order functions, and immutability.

Traits

Code reuse through traits for flexible composition.

Generics

Write type-safe generic code for maximum reusability.
