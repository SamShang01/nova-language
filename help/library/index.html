Welcome to Nova Standard Library

Nova's standard library provides a rich set of modules and functions to help you build applications efficiently. These modules cover common tasks like file I/O, networking, collections, date/time handling, and more.

How to Use the Standard Library

You can import standard library modules using the import keyword. For example: import std.io to use the I/O module.

Core Modules

  • std.io

    Input/output operations, including console I/O and file operations.

  • std.math

    Mathematical functions and constants.

  • std.string

    String manipulation functions.

  • std.collection

    Collection types and operations, including arrays, lists, dictionaries, and sets.

  • std.system

    System-related functions, including environment variables and process management.

Additional Modules

  • std.time

How to Use Library Modules

To use a library module, you need to import it first. Here's how:

Importing Modules

You can import modules in several ways:

  • import std.io - Imports the entire IO module
  • import std.io as io - Imports the IO module with an alias
  • from std.io import print, readLine - Imports specific functions from the IO module

Once imported, you can use the module's functions and types:

import std.io; // Using the print function from the IO module print("Hello, World!"); // Using the readLine function to get user input let name = readLine("Enter your name: "); print("Hello, " + name + "!");

Library Versioning

Nova's standard library follows a versioning scheme separate from the compiler. Each library module has its own version number, allowing for independent updates and improvements.

Library Versioning Scheme

Libraries use semantic versioning (MAJOR.MINOR.PATCH):

  • MAJOR: Breaking changes
  • MINOR: New features
  • PATCH: Bug fixes
