示例 泛型编程

泛型编程示例

学习如何使用泛型函数、泛型结构体和特质约束来编写可重用的代码

概述

泛型编程允许您编写适用于多种类型的代码，而无需为每种类型重复编写相同的逻辑。Nova 支持泛型函数、泛型结构体和特质约束。

泛型函数

使用类型参数创建通用函数

泛型结构体

定义可存储任意类型的容器

特质约束

限制泛型类型必须实现的功能

示例代码 

// 泛型函数示例
generic func identity<T>(value: T) -> T {
    return value;
}

generic func swap<T>(a: T, b: T) -> (T, T) {
    return (b, a);
}

generic func max<T>(a: T, b: T) -> T
    where T: Comparable
{
    if a > b {
        return a;
    } else {
        return b;
    }
}

// 泛型结构体示例
generic struct Pair<T, U> {
    first: T;
    second: U;
}

generic struct Stack<T> {
    items: [T];
    
    generic func push(self: Stack<T>, item: T) -> Stack<T> {
        self.items.push(item);
        return self;
    }
    
    generic func pop(self: Stack<T>) -> T {
        return self.items.pop();
    }
}

// 使用示例
let int_value = identity(42);
let string_value = identity("hello");

let pair = Pair<int, string>(first: 1, second: "hello");

let stack = Stack<int>(items: []);
stack.push(1);
stack.push(2);
print(stack.pop()); // 输出: 2

关键概念

类型参数

使用 <T> 语法定义类型参数。T 是一个占位符，在使用时会被实际类型替换。 可以定义多个类型参数，如 <T, U, V>。

特质约束

使用 where 子句限制泛型类型必须实现的特质。这确保了泛型代码只能使用特质提供的方法。

类型推断

编译器通常可以推断泛型类型参数，无需显式指定。例如 identity(42) 会自动推断 T 为 int。
