语法

Nova 的语法设计简洁明了，易于学习和使用。以下是基本的语法规则：

基本结构

import std.io;

fn main() {
    print("Hello, World!");
}

缩进

Nova 使用 4 个空格或 1 个制表符进行缩进，推荐使用空格。

注释

// 单行注释

/*
多行注释
多行注释
*/

类型系统

Nova 是一种静态类型语言，具有强大的类型系统。

基本类型

  • bool 布尔值：true 或 false
  • int 整数
  • float 浮点数
  • string 字符串
  • char 字符

类型注解

let x: int = 42;
let name: string = "Nova";
let is_active: bool = true;

函数

函数是 Nova 中的基本构建块。

函数定义

fn add(a: int, b: int) -> int {
    return a + b;
}

// 函数调用
let result = add(1, 2);

默认参数

fn greet(name: string, greeting: string = "Hello") {
    print(greeting + ", " + name + "!");
}

// 调用
greet("World"); // 输出: Hello, World!
greet("Alice", "Hi"); // 输出: Hi, Alice!

类和对象

Nova 支持面向对象编程，包括类、继承和多态。

类定义

class Person {
    name: string;
    age: int;
    
    fn __init__(self, name: string, age: int) {
        self.name = name;
        self.age = age;
    }
    
    fn greet(self) {
        print("Hello, my name is " + self.name);
    }
}

// 创建对象
let person = Person("Alice", 30);
person.greet(); // 输出: Hello, my name is Alice

继承

class Employee extends Person {
    job_title: string;
    
    fn __init__(self, name: string, age: int, job_title: string) {
        super(name, age);
        self.job_title = job_title;
    }
    
    fn greet(self) {
        super.greet();
        print("I am a " + self.job_title);
    }
}

let employee = Employee("Bob", 25, "Engineer");
employee.greet();
// 输出:
// Hello, my name is Bob
// I am a Engineer

模块

模块允许您组织和复用代码。

模块定义

在 Nova 中，每个文件都是一个模块。您可以使用 export 关键字导出函数、类或变量。

// math.nova
export fn add(a: int, b: int) -> int {
    return a + b;
}

export fn subtract(a: int, b: int) -> int {
    return a - b;
}

模块导入

// main.nova
import math;

let result = math.add(1, 2);
print(result); // 输出: 3

// 或使用别名
import math as m;
let result2 = m.subtract(5, 3);
print(result2); // 输出: 2

// 或导入特定函数
from math import add;
let result3 = add(10, 20);
print(result3); // 输出: 30

集合

Nova 提供了多种集合类型，包括数组、列表、字典和集合。

数组

let numbers: [int] = [1, 2, 3, 4, 5];
print(numbers[0]); // 输出: 1
numbers[1] = 10;
print(numbers[1]); // 输出: 10

字典

let person: {string: any} = {
    "name": "Alice",
    "age": 30,
    "is_active": true
};

print(person["name"]); // 输出: Alice
person["age"] = 31;
print(person["age"]); // 输出: 31

控制流

Nova 提供了多种控制流语句，包括条件语句和循环。

条件语句

let x = 10;

if x > 0 {
    print("x 是正数");
} elif x < 0 {
    print("x 是负数");
} else {
    print("x 是零");
}

循环

// for 循环
for i in 0..5 {
    print(i);
}

// while 循环
let i = 0;
while i < 5 {
    print(i);
    i += 1;
}

错误处理

Nova 使用 try-catch 语句进行错误处理。

try-catch

try {
    // 可能抛出错误的代码
    let result = 10 / 0;
} catch e {
    // 处理错误
    print("发生错误: " + e.message);
}

泛型

泛型允许您编写更通用和灵活的代码。

泛型函数

fn identity(value: T) -> T {
    return value;
}

let result = identity(42); // result 是 int 类型
let result2 = identity("Hello"); // result2 是 string 类型

泛型类

class Box {
    value: T;
    
    fn __init__(self, value: T) {
        self.value = value;
    }
    
    fn get(self) -> T {
        return self.value;
    }
}

let int_box = Box(42);
let string_box = Box("Hello");

特质

特质允许您定义方法签名，这些方法可以被类实现。

特质定义与实现

trait Printable {
    fn print(self);
}

class Person {
    name: string;
    
    fn __init__(self, name: string) {
        self.name = name;
    }
}

// 实现特质
impl Printable for Person {
    fn print(self) {
        print("Person: " + self.name);
    }
}

let person = Person("Alice");
person.print(); // 输出: Person: Alice

操作符

Nova 支持多种操作符，包括算术、比较、逻辑等。

算术操作符

  • + 加法
  • - 减法
  • * 乘法
  • / 除法
  • % 取模

比较操作符

  • == 等于
  • != 不等于
  • < 小于
  • > 大于
  • <= 小于等于
  • >= 大于等于

关键字

Nova 保留了以下关键字，不能用作标识符。

fn class extends impl trait import from export if elif else for while try catch return super self let const true false null as in

关于 Nova

Nova 是一种现代的静态类型编程语言，专为构建可靠和高效的软件而设计。
