欢迎使用 Nova 标准库

Nova 的标准库提供了丰富的模块和函数集，帮助您高效地构建应用程序。这些模块涵盖了文件 I/O、网络、集合、日期/时间处理等常见任务。

如何使用标准库

您可以使用 import 关键字导入标准库模块。例如：import std.io 以使用 I/O 模块。

核心模块

std.io

输入/输出操作，包括控制台 I/O 和文件操作。

std.math

数学函数和常量。

std.string

字符串操作函数。

std.collection

集合类型和操作，包括数组、列表、字典和集合。

std.system

系统相关函数，包括环境变量和进程管理。

附加模块

std.network

网络功能，包括 HTTP、TCP 和 UDP 操作。

std.time

日期和时间处理函数。

std.json

JSON 解析和序列化函数。

std.regex

正则表达式操作。

std.crypto

加密函数和工具。

std.thread

线程和并发工具。

std.random

随机数生成。

std.path

文件路径操作函数。

高级模块

std.async

异步编程工具。

std.stream

流处理工具。

std.compression

数据压缩和解压缩函数。

std.database

数据库连接工具。

std.web

Web 相关工具，包括 HTTP 服务器和客户端。

工具

std.assert

用于测试和调试的断言工具。

std.logger

日志工具。

std.format

字符串格式化工具。

std.encoding

字符编码工具。

如何使用库模块

要使用库模块，您需要先导入它。方法如下：

导入模块

您可以通过几种方式导入模块：

  • import std.io - 导入整个 IO 模块
  • import std.io as io - 导入 IO 模块并使用别名
  • from std.io import print, readLine - 从 IO 模块导入特定函数

导入后，您可以使用模块的函数和类型：

import std.io; // 使用 IO 模块中的 print 函数 print("Hello, World!"); // 使用 readLine 函数获取用户输入 let name = readLine("请输入您的名字："); print("你好，" + name + "！");

库版本管理

Nova 的标准库遵循与编译器分离的版本管理方案。每个库模块都有自己的版本号，允许独立更新和改进。

库版本管理方案

库使用语义化版本（MAJOR.MINOR.PATCH）：

  • MAJOR：重大更改
  • MINOR：新功能
  • PATCH：错误修复

为标准库做贡献

Nova 标准库开放贡献。如果您有新模块的想法或对现有模块的改进，可以通过 GitHub 存储库进行贡献。

在贡献之前，请查看贡献指南，并确保您的代码遵循项目的风格和质量标准。
