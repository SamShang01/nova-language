变量

在 Nova 中声明和使用变量

首页 > 教程 > 变量

变量声明

在 Nova 中，你可以使用 let 关键字来声明变量。变量是用于存储数据的命名容器。

// 声明变量 let message = "Hello"; let count = 42; let pi = 3.14; let isActive = true;

在上面的示例中，我们声明了四个变量：

  • message - 存储字符串 "Hello"
  • count - 存储整数 42
  • pi - 存储浮点数 3.14
  • isActive - 存储布尔值 true

类型注解

Nova 是静态类型语言，这意味着变量有固定的类型。你可以显式指定变量的类型，也可以让编译器自动推断类型。

// 显式类型注解 let name: string = "John"; let age: int = 30; let height: float = 1.75; let isStudent: bool = false; // 类型推断 let city = "New York"; // 自动推断为 string 类型 let population = 8_000_000; // 自动推断为 int 类型

变量可变性

默认情况下，Nova 中的变量是不可变的（常量）。如果你需要修改变量的值，你需要使用 var 关键字来声明可变变量。

// 不可变变量（常量） let constantValue = 10; // constantValue = 20; // 错误：不能修改不可变变量 // 可变变量 var mutableValue = 10; mutableValue = 20; // 正确：可以修改可变变量

作用域

变量的作用域决定了它在代码中的可见性和生命周期。在 Nova 中，变量的作用域由声明它的代码块决定。

// 全局作用域 let globalVar = "global"; def main() { // 函数作用域 let functionVar = "function"; if (true) { // 块作用域 let blockVar = "block"; print(globalVar); // 可以访问全局变量 print(functionVar); // 可以访问函数变量 print(blockVar); // 可以访问块变量 } print(globalVar); // 可以访问全局变量 print(functionVar); // 可以访问函数变量 // print(blockVar); // 错误：blockVar 不在作用域内 }

命名约定

Nova 推荐使用以下命名约定：

  • 变量和函数：使用驼峰命名法（camelCase），例如 userName、calculateTotal
  • 常量：使用全大写和下划线，例如 MAX_VALUE、PI
  • 类：使用 Pascal 命名法（PascalCase），例如 User、OrderItem
// 变量命名示例 let userName = "Alice"; let itemCount = 5; // 常量命名示例 let MAX_ITEMS = 100; let PI = 3.14159; // 函数命名示例 def calculateTotal(price: float, quantity: int) -> float { return price * quantity; }

类型转换

在 Nova 中，你可以在不同类型之间进行转换。有些转换是自动的（隐式转换），而其他转换需要显式指定。

// 隐式转换 let intValue: int = 42; let floatValue: float = intValue; // 自动转换 int 到 float // 显式转换 let floatNum: float = 3.14; let intNum: int = int(floatNum); // 显式转换 float 到 int let num: int = 123; let str: string = string(num); // 显式转换 int 到 string let boolValue: bool = true; let intBool: int = int(boolValue); // true 转换为 1，false 转换为 0

变量操作

你可以对变量执行各种操作，具体取决于它们的类型。

算术操作

let a = 10; let b = 5; let sum = a + b; // 15 let difference = a - b; // 5 let product = a * b; // 50 let quotient = a / b; // 2 let remainder = a % b; // 0

字符串操作

let firstName = "John"; let lastName = "Doe"; let fullName = firstName + " " + lastName; // "John Doe" let greeting = "Hello, " + firstName + "!"; // "Hello, John!"

赋值操作

var x = 10; x += 5; // 等同于 x = x + 5 x -= 3; // 等同于 x = x - 3 x *= 2; // 等同于 x = x * 2 x /= 4; // 等同于 x = x / 4

常量

除了使用 let 声明的不可变变量外，Nova 还支持使用 const 关键字声明常量。常量必须在声明时初始化，并且其值在整个程序执行过程中不能改变。

// 声明常量 const PI = 3.14159; const MAX_USERS = 1000; const APP_NAME = "Nova App"; // PI = 3.14; // 错误：常量不能修改

练习

尝试完成以下练习，以巩固你对变量的理解：

  1. 声明一个名为 name 的变量，存储你的名字
  2. 声明一个名为 age 的变量，存储你的年龄
  3. 声明一个名为 isStudent 的变量，存储一个布尔值表示你是否是学生
  4. 使用这些变量创建一个问候语字符串，例如 "Hello, John! You are 25 years old and you are a student."
  5. 打印这个问候语
// 练习答案示例 def main() { let name = "John"; let age = 25; let isStudent = true; let greeting = "Hello, " + name + "! You are " + string(age) + " years old and you are " + (isStudent ? "a student" : "not a student") + "."; print(greeting); }
