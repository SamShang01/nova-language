泛型

在 Nova 中使用泛型实现类型安全的代码

首页 > 教程 > 泛型

泛型简介

泛型允许您编写可以处理多种类型的代码，同时保持类型安全性。它们使您能够创建可重用的组件，这些组件可以在维护编译时类型检查的同时操作不同类型的数据。

泛型函数

您可以通过在尖括号中指定类型参数（<T>）来定义泛型函数：

// 泛型函数 def printArray(arr: Array) { for (let item in arr) { print(item); } } // 用于不同类型 let numbers = [1, 2, 3, 4, 5]; let strings = ["hello", "world", "nova"]; printArray(numbers); // 输出: 1, 2, 3, 4, 5 printArray(strings); // 输出: hello, world, nova

多个类型参数

您可以定义具有多个类型参数的函数：

// 多个类型参数 def pair(first: T, second: U) -> Array { return [first, second]; } // 使用 let result1 = pair(1, "hello"); print(result1); // 输出: [1, "hello"] let result2 = pair(true, 3.14); print(result2); // 输出: [true, 3.14]

泛型类

您可以通过在尖括号中指定类型参数来定义泛型类：

// 泛型类 class Box { private let value: T; def __init__(self, value: T) { self.value = value; } def getValue() -> T { return self.value; } def setValue(value: T) { self.value = value; } } // 用于不同类型 let intBox = Box(42); print(intBox.getValue()); // 输出: 42 intBox.setValue(100); print(intBox.getValue()); // 输出: 100 let stringBox = Box("Hello"); print(stringBox.getValue()); // 输出: Hello stringBox.setValue("World"); print(stringBox.getValue()); // 输出: World

类中的多个类型参数

类也可以有多个类型参数：

// 类中的多个类型参数 class Pair { let first: T; let second: U; def __init__(self, first: T, second: U) { self.first = first; self.second = second; } def getFirst() -> T { return self.first; } def getSecond() -> U { return self.second; } } // 使用 let pair1 = Pair(1, "one"); print(pair1.getFirst()); // 输出: 1 print(pair1.getSecond()); // 输出: one let pair2 = Pair("test", true); print(pair2.getFirst()); // 输出: test print(pair2.getSecond()); // 输出: true

泛型接口和特性

您可以定义泛型接口和特性：

// 泛型特性 trait Container { def add(item: T); def remove() -> T; def isEmpty() -> bool; } // 实现泛型特性 class Stack implements Container { private let items: Array = []; def add(item: T) { self.items.push(item); } def remove() -> T { if (self.isEmpty()) { throw new Error("Stack is empty"); } return self.items.pop(); } def isEmpty() -> bool { return self.items.length == 0; } } // 使用 let stack = Stack(); stack.add(1); stack.add(2); stack.add(3); print(stack.remove()); // 输出: 3 print(stack.remove()); // 输出: 2 print(stack.remove()); // 输出: 1 print(stack.isEmpty()); // 输出: true

类型约束

您可以为类型参数添加约束，以确保它们满足特定要求：

// 类型约束 trait Comparable { def compareTo(other: any) -> int; } class Person implements Comparable { let name: string; let age: int; def __init__(self, name: string, age: int) { self.name = name; self.age = age; } def compareTo(other: any) -> int { if (other is Person) { return self.age - other.age; } return 0; } } // 带约束的泛型函数 def findMax(items: Array) -> T { if (items.length == 0) { throw new Error("Array is empty"); } let max = items[0]; for (let item in items) { if (item.compareTo(max) > 0) { max = item; } } return max; } // 使用 let people = [ Person("John", 30), Person("Alice", 25), Person("Bob", 35) ]; let oldest = findMax(people); print(oldest.name); // 输出: Bob

默认类型参数

您可以为类型参数指定默认类型：

// 默认类型参数 def defaultValue() -> T { if (T is string) { return "default" as T; } elif (T is int) { return 0 as T; } elif (T is bool) { return false as T; } return null as T; } // 使用 print(defaultValue()); // 输出: default print(defaultValue()); // 输出: 0 print(defaultValue()); // 输出: false print(defaultValue()); // 输出: default (使用默认的 string 类型)

非泛型类中的泛型方法

您可以在非泛型类中定义泛型方法：

// 非泛型类中的泛型方法 class Utils { def static printList(list: Array) { for (let item in list) { print(item); } } def static map(list: Array, mapper: (T) -> U) -> Array { let result: Array = []; for (let item in list) { result.push(mapper(item)); } return result; } } // 使用 let numbers = [1, 2, 3, 4, 5]; Utils.printList(numbers); // 输出: 1, 2, 3, 4, 5 let doubled = Utils.map(numbers, (x) => x * 2); Utils.printList(doubled); // 输出: 2, 4, 6, 8, 10 let strings = Utils.map(numbers, (x) => "Number: " + x); Utils.printList(strings); // 输出: Number: 1, Number: 2, Number: 3, Number: 4, Number: 5

通配符类型

您可以使用通配符类型来表示未知类型：

// 通配符类型 class Box { private let value: T; def __init__(self, value: T) { self.value = value; } def getValue() -> T { return self.value; } } // 使用通配符类型 def printBoxContent(box: Box) { print(box.getValue()); } // 使用 let intBox = Box(42); let stringBox = Box("Hello"); printBoxContent(intBox); // 输出: 42 printBoxContent(stringBox); // 输出: Hello

类型擦除

在 Nova 中，泛型是使用类型擦除实现的，这意味着类型参数在运行时被移除。这允许高效的代码生成，同时保持编译时类型安全。

注意

由于类型擦除，您不能在泛型函数或类内部使用类型参数进行运行时类型检查，如 typeof T 或 is T。

最佳实践

  • 使用描述性的类型参数名称：对于简单情况使用有意义的名称如 T、U、V，或对于复杂情况使用更具描述性的名称如 ElementType。
  • 在必要时添加类型约束：使用约束确保类型参数满足特定要求。
  • 保持泛型简单：不要使用太多类型参数使代码过于复杂。
  • 为集合类型使用泛型：泛型对于列表、栈和队列等集合类特别有用。
  • 记录泛型代码：添加注释解释如何使用泛型类型。

常见错误

  • 忘记指定类型参数：创建泛型类的实例时始终指定类型参数。
  • 在运行时检查中使用类型参数：记住类型参数在运行时被擦除。
  • 过度使用泛型：当简单的函数或类足够时，不要使用泛型。
  • 忽略类型约束：使用受约束的类型参数时，确保实现所有必需的方法。
  • 混淆类型参数：注意多个类型参数的顺序。
