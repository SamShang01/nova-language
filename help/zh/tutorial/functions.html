函数

在 Nova 中定义和使用函数

首页 > 教程 > 函数

函数定义

在 Nova 中，你可以使用 def 关键字来定义函数。函数是一段可重用的代码块，用于执行特定的任务。

// 定义一个简单的函数 def greet() { print("Hello, World!"); } // 调用函数 greet(); // 输出: Hello, World!

函数参数

函数可以接受参数，这些参数是在函数调用时传递给函数的值。

// 带参数的函数 def greet(name: string) { print("Hello, " + name + "!"); } // 调用函数 greet("John"); // 输出: Hello, John!

你可以定义多个参数：

// 多个参数 def add(a: int, b: int) { print(a + b); } add(5, 3); // 输出: 8

返回值

函数可以使用 return 语句返回值。

// 带返回值的函数 def add(a: int, b: int) -> int { return a + b; } let result = add(5, 3); print(result); // 输出: 8

返回类型注解是可选的，但推荐使用，因为它可以提供更好的类型安全性。

默认参数

你可以为函数参数指定默认值，这样在调用函数时可以省略这些参数。

// 带默认参数的函数 def greet(name: string = "World") { print("Hello, " + name + "!"); } // 调用函数 greet(); // 输出: Hello, World! greet("John"); // 输出: Hello, John!

变量参数

你可以使用 ... 语法来定义接受可变数量参数的函数。

// 变量参数 def sum(...numbers: int) -> int { let total = 0; for (let number in numbers) { total += number; } return total; } print(sum(1, 2, 3)); // 输出: 6 print(sum(4, 5, 6, 7)); // 输出: 22

递归函数

函数可以调用自身，这称为递归。

// 递归函数计算阶乘 def factorial(n: int) -> int { if (n <= 1) { return 1; } return n * factorial(n - 1); } print(factorial(5)); // 输出: 120

匿名函数

你可以使用 lambda 关键字创建匿名函数（也称为 lambda 函数）。

// 匿名函数 let add = lambda a, b => a + b; print(add(5, 3)); // 输出: 8 // 作为参数传递 let numbers = [1, 2, 3, 4, 5]; let doubled = numbers.map(lambda x => x * 2); print(doubled); // 输出: [2, 4, 6, 8, 10]

函数作为值

在 Nova 中，函数可以作为值存储在变量中，也可以作为参数传递给其他函数。

// 函数作为值 let greet = def(name: string) { print("Hello, " + name + "!"); }; greet("John"); // 输出: Hello, John! // 函数作为参数 def apply(f, value) { return f(value); } let double = lambda x => x * 2; print(apply(double, 5)); // 输出: 10

函数组合

你可以组合多个函数来创建新的函数。

// 函数组合 let addOne = lambda x => x + 1; let double = lambda x => x * 2; // 组合函数 let doubleAndAddOne = lambda x => addOne(double(x)); print(doubleAndAddOne(5)); // 输出: 11

作用域

函数内部声明的变量只在函数内部可见，这称为局部作用域。

let globalVar = "global"; def test() { let localVar = "local"; print(globalVar); // 可以访问全局变量 print(localVar); // 可以访问局部变量 } test(); // print(localVar); // 错误：localVar 不在作用域内

闭包

闭包是一个函数，它可以访问其词法作用域中的变量，即使在其定义的作用域之外执行。

// 闭包 def makeCounter() { var count = 0; return def() { count += 1; return count; }; } let counter = makeCounter(); print(counter()); // 输出: 1 print(counter()); // 输出: 2 print(counter()); // 输出: 3

练习

尝试完成以下练习，以巩固你对函数的理解：

  1. 编写一个函数，计算两个数的乘积
  2. 编写一个函数，检查一个数是否为偶数
  3. 编写一个递归函数，计算斐波那契数列的第 n 项
  4. 编写一个函数，接受一个字符串并返回其反转形式
// 练习答案示例 // 1. 计算两个数的乘积 def multiply(a: int, b: int) -> int { return a * b; } // 2. 检查一个数是否为偶数 def isEven(n: int) -> bool { return n % 2 == 0; } // 3. 计算斐波那契数列的第 n 项 def fibonacci(n: int) -> int { if (n <= 1) { return n; } return fibonacci(n - 1) + fibonacci(n - 2); } // 4. 反转字符串 def reverse(s: string) -> string { let result = ""; for (let i = s.length - 1; i >= 0; i--) { result += s[i]; } return result; }
