Hello World

你的第一个 Nova 程序

首页 > 教程 > Hello World

介绍

欢迎来到你的第一个 Nova 程序！在本教程中，你将学习如何创建和运行一个简单的 "Hello, World!" 程序。这是学习任何编程语言的传统第一步。

步骤 1：创建 Nova 文件

首先，创建一个带有 .nova 扩展名的新文件。你可以随意命名，但在本教程中，我们称之为 hello.nova。

Windows

  1. 打开记事本或你喜欢的文本编辑器
  2. 将文件保存为 hello.nova（确保在保存对话框中选择 "所有文件"）

macOS/Linux

  1. 打开你喜欢的文本编辑器（例如 VS Code、Sublime Text 或 nano）
  2. 将文件保存为 hello.nova

步骤 2：编写 Hello World 程序

现在，让我们为 Hello World 程序编写代码。在 Nova 中，向控制台打印内容最简单的方法是使用 print 函数。

def main() { print("Hello, World!"); }

让我们分解一下这段代码的作用：

  • def main() { - 这定义了一个名为 main 的函数。在 Nova 中，main 函数是程序的入口点。
  • print("Hello, World!"); - 这调用了 print 函数，以字符串 "Hello, World!" 作为参数。这将在控制台上显示文本。
  • } - 这关闭了 main 函数。

步骤 3：运行程序

现在我们已经编写了程序，让我们运行它。打开终端或命令提示符，导航到你保存 hello.nova 的目录。

运行程序

# 使用 nova 命令运行程序 nova hello.nova

你应该看到输出：

Hello, World!

替代方案：使用 REPL

你也可以使用 REPL（读取-求值-打印循环）交互式运行 Nova 代码。要启动 REPL，只需在终端中输入 nova：

$ nova Nova 1.0.0 (REPL mode) Type 'exit' to quit. > print("Hello, World!"); Hello, World! >

要退出 REPL，输入 exit 或按 Ctrl+D（Windows 上按 Ctrl+Z）。

理解代码

让我们更仔细地看一下 Hello World 程序：

函数定义

在 Nova 中，函数使用 def 关键字定义，后跟函数名、参数括号和用花括号括起来的代码块。

main 函数

main 函数在 Nova 中是特殊的。当你运行 Nova 程序时，解释器会查找 main 函数并自动调用它。这是程序的入口点。

print 函数

print 函数是 Nova 中的内置函数，用于向控制台显示文本。它接受一个或多个参数，并以空格分隔打印它们。

变体

以下是 Hello World 程序的一些变体，你可以尝试：

打印多行

def main() { print("Hello, World!"); print("欢迎使用 Nova!"); }

使用变量

def main() { let message = "Hello, World!"; print(message); }

使用变量打印

def main() { let name = "World"; print("Hello, " + name + "!"); }

常见错误

以下是你在编写第一个 Nova 程序时可能遇到的一些常见错误：

  • 语法错误：忘记关闭括号或花括号。确保所有打开的括号、方括号和花括号都有相应的关闭符号。
  • 名称错误：使用尚未定义的变量或函数。检查你的拼写，确保所有名称都正确。
  • 文件未找到：从错误的目录运行程序。确保你在与 .nova 文件相同的目录中。
