控制流

在 Nova 中使用条件语句和循环

首页 > 教程 > 控制流

条件语句

条件语句允许你根据条件执行不同的代码块。

if 语句

let x = 10; if (x > 0) { print("x 是正数"); }

if-else 语句

let x = -5; if (x > 0) { print("x 是正数"); } else { print("x 不是正数"); }

if-else if-else 语句

let x = 0; if (x > 0) { print("x 是正数"); } else if (x < 0) { print("x 是负数"); } else { print("x 是零"); }

switch 语句

switch 语句允许你根据表达式的值执行不同的代码块。

let day = 3; switch (day) { case 1: print("星期一"); break; case 2: print("星期二"); break; case 3: print("星期三"); break; case 4: print("星期四"); break; case 5: print("星期五"); break; default: print("周末"); break; }

循环

循环允许你重复执行代码块。

for 循环

// 基本 for 循环 for (let i = 0; i < 5; i++) { print(i); } // 输出: 0, 1, 2, 3, 4

for-in 循环

for-in 循环用于遍历数组或集合中的元素。

let fruits = ["苹果", "香蕉", "橙子"]; for (let fruit in fruits) { print(fruit); } // 输出: 苹果, 香蕉, 橙子

while 循环

let i = 0; while (i < 5) { print(i); i++; } // 输出: 0, 1, 2, 3, 4

do-while 循环

do-while 循环至少执行一次代码块，然后根据条件决定是否继续执行。

let i = 0; do { print(i); i++; } while (i < 5); // 输出: 0, 1, 2, 3, 4

循环控制语句

break 语句

break 语句用于跳出循环。

for (let i = 0; i < 10; i++) { if (i == 5) { break; } print(i); } // 输出: 0, 1, 2, 3, 4

continue 语句

continue 语句用于跳过当前循环的剩余部分，进入下一次循环。

for (let i = 0; i < 10; i++) { if (i % 2 == 0) { continue; } print(i); } // 输出: 1, 3, 5, 7, 9

逻辑运算符

逻辑运算符用于组合条件。

// 与运算符 && let x = 10; if (x > 0 && x < 20) { print("x 在 0 和 20 之间"); } // 或运算符 || let y = 25; if (y < 0 || y > 20) { print("y 不在 0 和 20 之间"); } // 非运算符 ! let z = true; if (!z) { print("z 是 false"); } else { print("z 是 true"); }

比较运算符

比较运算符用于比较值。

let a = 10; let b = 5; print(a == b); // 等于: false print(a != b); // 不等于: true print(a > b); // 大于: true print(a < b); // 小于: false print(a >= b); // 大于等于: true print(a <= b); // 小于等于: false

三元运算符

三元运算符是一种简洁的条件表达式。

let x = 10; let result = x > 0 ? "正数" : "非正数"; print(result); // 输出: 正数

嵌套控制流

你可以在一个控制流语句中嵌套另一个控制流语句。

for (let i = 0; i < 5; i++) { if (i % 2 == 0) { print(i + " 是偶数"); } else { print(i + " 是奇数"); } }

练习

尝试完成以下练习，以巩固你对控制流的理解：

  1. 编写一个程序，使用 for 循环打印 1 到 10 的数字
  2. 编写一个程序，使用 while 循环计算 1 到 100 的和
  3. 编写一个程序，使用 if-else 语句检查一个数是否为素数
  4. 编写一个程序，使用 switch 语句将数字转换为对应的星期几
// 练习答案示例 // 1. 打印 1 到 10 的数字 def printNumbers() { for (let i = 1; i <= 10; i++) { print(i); } } // 2. 计算 1 到 100 的和 def calculateSum() { let sum = 0; let i = 1; while (i <= 100) { sum += i; i++; } print("1 到 100 的和是: " + sum); } // 3. 检查一个数是否为素数 def isPrime(n: int) -> bool { if (n <= 1) { return false; } for (let i = 2; i < n; i++) { if (n % i == 0) { return false; } } return true; } // 4. 将数字转换为对应的星期几 def getDayOfWeek(day: int) -> string { switch (day) { case 1: return "星期一"; case 2: return "星期二"; case 3: return "星期三"; case 4: return "星期四"; case 5: return "星期五"; case 6: return "星期六"; case 7: return "星期日"; default: return "无效的星期几"; } }
