安装指南

在您的系统上安装 Nova

首页 > 教程 > 安装

系统要求

Nova 设计用于在多种系统上运行。以下是最低要求：

  • 操作系统：Windows 10+，macOS 10.15+，Linux（Ubuntu 18.04+，Fedora 30+）
  • CPU：1 GHz 或更快的处理器
  • 内存：2 GB RAM
  • 存储空间：100 MB 可用空间
  • Python：3.7 或更高版本（运行时必需）

安装方法

Windows 安装

  1. 下载安装程序：访问 Nova 网站并下载适合您系统架构（32位或 64位）的 Windows 安装程序（.exe）。
  2. 运行安装程序：双击下载的 .exe 文件并按照屏幕上的说明操作。
  3. 添加到 PATH：在安装过程中，选择将 Nova 添加到系统 PATH 的选项。
  4. 验证安装：打开新的命令提示符并运行 nova --version 以确认安装成功。

macOS 安装

  1. 下载包：访问 Nova 网站并下载 macOS 包（.pkg）。
  2. 运行安装程序：双击下载的 .pkg 文件并按照屏幕上的说明操作。
  3. 添加到 PATH：安装程序会自动将 Nova 添加到您的 PATH。
  4. 验证安装：打开新的终端窗口并运行 nova --version 以确认安装成功。

Linux 安装

使用包管理器（Debian/Ubuntu）
# 添加 Nova 仓库 sudo apt update sudo apt install wget gnupg wget -qO- https://nova-lang.org/keys/nova.gpg | sudo apt-key add - echo "deb https://nova-lang.org/repos/deb stable main" | sudo tee /etc/apt/sources.list.d/nova.list # 安装 Nova sudo apt update sudo apt install nova
使用包管理器（Fedora）
# 添加 Nova 仓库 sudo dnf config-manager --add-repo https://nova-lang.org/repos/rpm/nova.repo # 安装 Nova sudo dnf install nova
从源代码安装
  1. 克隆仓库： git clone https://github.com/nova-lang/nova.git
  2. 导航到目录： cd nova
  3. 安装依赖： pip install -r requirements.txt
  4. 安装 Nova： python setup.py install
  5. 验证安装： nova --version

替代安装方法

使用 pip

您也可以使用 pip 安装 Nova：

pip install nova-lang

从源代码

对于想要从源代码构建的高级用户：

  1. 克隆仓库： git clone https://github.com/nova-lang/nova.git
  2. 导航到目录： cd nova
  3. 安装依赖： pip install -r requirements.txt
  4. 构建 Nova： python setup.py build
  5. 安装 Nova： python setup.py install

验证安装

要验证 Nova 是否已正确安装，请打开终端或命令提示符并运行：

nova --version

您应该看到 Nova 的版本号显示。您还可以通过输入以下命令运行 REPL（读取-求值-打印循环）：

nova

这将启动 Nova 交互式 shell，您可以在其中测试简单的表达式。

故障排除

常见问题

  • 找不到 Nova 命令：确保 Nova 已添加到您的系统 PATH。在 Windows 上，您可能需要重新启动命令提示符。在 Linux/macOS 上，您可能需要运行 source ~/.bashrc 或 source ~/.zshrc。
  • Python 依赖问题：确保您已安装 Python 3.7 或更高版本。您可以使用 python --version 检查您的 Python 版本。
  • 权限错误：在 Linux/macOS 上，您可能需要在安装 Nova 时使用 sudo。
